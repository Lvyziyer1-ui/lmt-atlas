UC2 Engineering Gain

This panel compares wild-type and engineered constructs within the same parent background, so the visual question is whether engineering shifted tolerance beyond source-to-source noise. Read it as paired evidence: lower LTS_ref means stronger methanol tolerance, while wide intervals warn that an apparent gain may not be reliable.

UC3 Tolerance Ranking Landscape

This panel orders construct-level records by the frozen tolerance score and overlays the rank-stability tier. Read it as a map of the 439 scored constructs: only entries that stay near the tolerant or fragile end after sensitivity checks receive a robust tier, while most remain uncertain or intermediate.

Reproducibility Ceiling

This panel asks how much agreement can be expected when independent sources report related methanol-challenge measurements. Read it as the reproducibility ceiling for heterogeneous literature evidence: measurement rows can support patterns, but unmatched protocols should not be treated as interchangeable replicates.

Rank-Stability Tier

The methanol-tolerance badge is a confidence coarse tier based on rank stability, not an absolute credible-interval call: More tolerant 37, Less tolerant 38, Uncertain 364.

The public methanol-tolerance badge is a coarse rank-stability tier derived from frozen construct LTS_ref and independent rank-stability checks. It complements the numeric score and is not an absolute credible-interval class.
