Rank-Stability Tier

The methanol-tolerance badge is a confidence coarse tier based on rank stability, not an absolute credible-interval call: More tolerant 37, Less tolerant 38, Uncertain 364.

The public methanol-tolerance badge is a coarse rank-stability tier derived from frozen construct LTS_ref, Stage 5 sensitivity stability, and the MCMC cross-check. It complements the numeric score and is not an absolute credible-interval class.
