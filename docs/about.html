About LMT-Atlas

LMT-Atlas is a curated, methanol-tolerance-specific evidence database for TAG lipases, designed to keep enzyme identity, construct boundaries, assay protocols, measurements, and provenance auditable.

Project Scope

The database focuses on intrinsic methanol-tolerance evidence and does not treat reactor-scale biodiesel process performance as the primary tolerance label. Documentation and the data dictionary describe the evidence tiers and source-count scopes.

Because assay conditions are heterogeneous, users should filter by incubation temperature, assay method/substrate, methanol concentration, and exposure time before comparing entries. Cross-condition raw residual activity values are not directly comparable.

Browse/Search exports support condition-filtered comparable subsets plus optional tiered labels in conservative mode; exported classification labels are temporary training views derived from frozen scores, not replacement official labels.

The public tolerance badge is a confidence coarse tier with 37 More tolerant, 38 Less tolerant, and 365 Uncertain constructs; it complements continuous score values rather than replacing them.

Known limitation: LipA9 is an incremental v2.0 entry with low evidence support, a wide interval, and digitized Fig. 4b values that need review.
