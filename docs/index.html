Lipase Methanol Tolerance Atlas

A curated evidence database for enzyme-intrinsic methanol tolerance in TAG lipases, linking canonical enzyme identities, constructs, assay protocols, quantitative measurements, and literature/patent provenance.

Release note: Stable v1.0 static release with frozen LTS_ref values, source-scope downloads, and reader-facing evidence caveats.

How To Use This Database

Find candidate lipases

Start with Most tolerant first, then filter by confidence, evidence tier, organism, assay class, and structural family.

Compare only comparable assays

Use broad method and substrate classes before selecting exact assay text; avoid comparing raw residual activity across unmatched conditions.

Open evidence details

Check identifiers, tolerance summary, measurements, conditions, and literature or patent provenance before reusing an entry.

Data Model

1. Enzyme SpeciesCanonical ID, species, EC, identifier status, structure readiness.
2. Variant / ConstructWT, mutant, engineered construct, parent-child boundary.
3. Assay ProtocolChallenge context, A0 quality, buffer, temperature, enzyme concentration.
4. MeasurementMethanol concentration, time, residual activity, extraction method.
5. SourceLiterature DOI, patent publication, figure/table anchor, caveat.

Explore By Task

Find an enzyme or construct

Search names, species, canonical IDs, UniProt, PDB, EC, DOI, and patent numbers.

Browse all canonical entries

Open the 625-row identity index with identifier-status caveats.

Inspect evidence matrices

Filter methanol concentration, incubation time, residual activity, source type, and confidence tier.

Download reusable tables

Use CSV/SQLite artifacts generated from the same release snapshot.

Scope And Citation

This v1.0 static release supports browsing, search, provenance inspection, and table download for the locked source-scope publication bundle.

Structural module overlays and family taxonomy claims remain outside this release. Official tolerance fields are active for the web publication scope.
